# InBody AI 건강관리 시스템

InBody 체성분 분석 결과를 기반으로 AI가 개인 맞춤형 건강관리 플랜을 제공하는 시스템입니다.

## 🚀 빠른 시작

1. 패키지 설치
```bash
pip install -r requirements.txt
```

2. .env 파일에 API 키 입력

3. 실행
```bash
streamlit run app.py
```

## 📋 주요 기능

- 📸 InBody 결과지 OCR (Google Vision API)
- 🤖 AI 건강 분석 (Perplexity AI)
- 🎯 하모니서치 알고리즘 기반 최적 운동 일정
- 📄 PDF 건강관리 레포트 생성

## 🛠 기술 스택

- **Frontend**: Streamlit
- **OCR**: Google Cloud Vision API
- **AI**: Perplexity AI
- **최적화**: Harmony Search Algorithm
- **PDF**: FPDF2
